Meal Recommendations Web App

This web app provides personalized meal recommendations based on your demographic information, physical activity, and dietary preferences.

Author: Chan Kwan Ho

Demographic Information

Collect and store user demographic information and physical activity levels.

Personalized Daily Recommendation Intake

Integrate a Basal Metabolic Rate (BMR) calculator to display Daily Recommended Intake (DRI) information.

Dietary Preferences

Allow users to input dietary preferences and provide meal recommendations.

Food Database

Implement a food database with tabs for sorting based on restaurant, taste, or food type.

Meal Record Tracking

Track and display the user's past meal records.
